Preview component allows to open modal for viewing some document and to work with it (download, zoom, rotate and etc)

As a document you can provide any content like images,

The component automatically adjusts to the mobile device Don't forget to add tui-preview-host to the tui-root of your application (more details on the "Setup" tab)
  1. Add TuiPreviewHostModule to main module:
  2. Add tui-preview-host to the tui-root:
  3. Import TuiPreviewModule to the module, witch will call the preview
  4. Create a template with tui-preview
  5. Open the component from the TuiPreviewService as in the example
